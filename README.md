# Homepage Configs

These are the example configuration files for the Homepage dashboard setup, as featured on the [CoreWorxLab](https://youtube.com/@coreworxlab) YouTube channel. 

Video link: [Homepage vs Homarr: The Best Dashboard for Your Home Server](https://www.youtube.com/watch?v=ATCTWXvoM38)

**Important:** Before using these files, you will need to replace the placeholder values for URLs, usernames, passwords, and API keys. These placeholders are indicated by double braces, like `{{ HOMEPAGE_VAR_SERVER_URL }}`, within the `.yaml` files.
